LAYHANDS.2DA

This file defines amount of HP healed by Paladin in respect to his level..

The first row (titles of columns ) is filled by the names of defined characteristics of PC (like MIN_STR = minimum strength).

The first column is filled by class name.

The first row is filled by numbers indicating level

The next row(s) contain of the number of HP healed by this ability.
