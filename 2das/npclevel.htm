NPCLEVEL.2DA

This file defines PNPC level when they are met first time by PC.

The first column is filled by NPCs death variables names

The first row is filled by PC's level in the moment of encountering NPC.

On the crossing of these two there are names of CRE files that are used for given NPC for given level of PC.
