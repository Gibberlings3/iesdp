Known values for 'Type' are:
0 Cumulative Modifier -> Morale = Morale + 'Statistic Modifier' value
1 Flat Value Modifier -> Morale = 'Statistic Modifier' value
2 Percentage Modifier -> Morale = (Morale * 'Statistic Modifier' value) / 100 {% capture note %} Any Morale-based effect (i.e. Panic) may be added or removed by this effect. {% endcapture %} {% include note.html %}
