--- n: 29 name: "Stat: Electricity Resistance Modifier" param1: "Statistic Modifier" param2: "Type" bg1: 0 bg2: 0 bgee: 0 iwd1: 0 iwd2: 1 pst: 0 --- Applies the modifier value specified by the 'Statistic Modifier' field in the style specified by the 'Type' field.
Known values for 'Type' are:
0 Cumulative Modifier -> Resistance = Resistance + 'Statistic Modifier' value
1 Flat Value Modifier -> Resistance = 'Statistic Modifier' value

'Statistic Modifier' can be positive or negative.
