--- n: 37 name: "Stat: Save vs. Spells Modifier" param1: "Statistic Modifier" param2: "Type" bg1: 1 bg2: 1 bgee: 1 iwd1: 1 iwd2: 0 pst: 1 --- Applies the modifier value specified by the 'Statistic Modifier' field in the style specified by the 'Type' field.
Known values for 'Type' are:
0 Cumulative Modifier -> Save = Save + 'Statistic Modifier' value
1 Flat Value Modifier -> Save = 'Statistic Modifier' value
2 Percentage Modifier -> Save = (Save * 'Statistic Modifier' value) / 100

'Statistic Modifier' can be positive or negative.
