--- n: 39 name: "State: Unconsciousness" param1: "Irrelevant" param2: "Wake on Damage?" special: "Icon" bg1: 1 bg2: 1 bgee: 1 iwd1: 1 iwd2: 0 pst: 0 --- Applies the unconsciousness effect to the targeted creature(s).
Known values for Wake on Damage? are (only in EEs):
  • 0 ⟶ Yes
  • 1 ⟶ No

Icon defines the index from STATDESC.2DA for custom portrait icons (EE only). Default value is Sleep.

This opcode sets the STATE_SLEEPING flag.

{% capture note %} Timing Modes 1/4/7 will set the STATE_HELPLESS flag. In such a case, this flag cannot be removed in-game.
Moreover, they will also set STATE_SLEEPING, which works as the Wake on Hit option. It's just not usable because of the irremovable STATE_HELPLESS. {% endcapture %} {% include note.html %}
