--- n: 39 name: "State: Unconsciousness" param1: "Irrelevant" param2: "Wake on Damage?" special: "Icon" bg1: 1 bg2: 1 bgee: 1 iwd1: 1 iwd2: 0 pst: 0 --- Applies the unconsciousness effect to the targeted creature(s).
Known values for Wake on Damage? are (only in EEs):
  • 0 ⟶ Yes
  • 1 ⟶ No

Icon defines the index from STATDESC.2DA for custom portrait icons (EE only). Default value is Sleep. If it is set, also spell state PRONE will be set. Otherwise the creature will be moved to the backlist and become untargettable by most scripts.

This opcode sets the STATE_SLEEPING and STATE_HELPLESS flag bits.

{% capture note %} In case of Timing Modes 1/4/7 the bits cannot be removed in-game.
Moreover, the STATE_SLEEPING works as the Wake on Hit option. It's just not usable because of the unremovable STATE_HELPLESS. {% endcapture %} {% include note.html %} {% capture note %} Immunity to this opcode does not prevent unconsciousness from stunning damage done by normal fists. {% endcapture %} {% include note.html %}
