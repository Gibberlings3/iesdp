To figure out the Animate Set parameter value, do the following:
Look up the value for the Animation Set you wish to use for this effect in the RESDATA.INI file (it'll be the number in the brackets directly above it), then do the following:
If you want the animation to be falsely/'clown' coloured, add 24576
If you want the animation to be true coloured, add 57344 {% capture note %} There are several animations that SHOULD NOT be used with this effect. They are special, and will result in a CRASH!
