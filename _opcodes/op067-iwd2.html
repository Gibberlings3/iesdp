--- n: 67 name: "Summon" param1: "Creature Number" param2: "Summoning Animation" bg1: 0 bg2: 0 bgee: 0 iwd1: 0 iwd2: 1 pst: 0 --- The Creature Number parameter set the number to be summoned, and the Summoning Animation parameter sets the type of animation to use for the summoning.
Known values for the Summoning Animation parameter are:
0 no animation
1 monster summoning circle
2 animal summoning circle
3 earth summoning circle
4 fire summoning circle
5 water summoning circle
7 puff of smoke only {% capture note %} The Dice values can be used to set the number of creatures summoned, in addition to the Number parameter {% endcapture %} {% include info.html %}
