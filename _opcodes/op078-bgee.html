--- n: 78 name: "State: Disease" param1: "Damage" param2: "Type" special: "Behaviour" bg1: 0 bg2: 0 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Set the Damage parameter as the amount of damage, and the Disease Type parameter as how the disease functions.
Known values for 'Type' are:
0 1 HP per second
1 1 HP per second
2 'Damage' per Second
3 Cause 1HP every 'Damage' seconds
4 Strength Penalty
5 Dexterity Penalty
6 Constitution Penalty
7 Intelligence Penalty
8 Wisdom Penalty
9 Charisma Penalty
10 Slow Effect
11 Mold touch/Single
12 Mold touch/Decrement
13 Contagion

'Damage' can be positive (penalty) or negative (bonus).
The field 'Behaviour' is used for "Mold touch" and consists of two word values:
First word: # rounds to apply the affect
Second word: Whether to use damage defined in 'Damage'. {% capture note %} Permanent timing modes require a duration for this effect to work. {% endcapture %} {% include important.html %}
