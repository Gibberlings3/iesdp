--- n: 98 name: "Regeneration" param1: "Regen/Time" param2: "Regen Type" bg1: 0 bg2: 0 bgee: 0 iwd1: 0 iwd2: 0 pst: 1 --- The Regen Type parameter sets how the Regen/Time parameter works.
0 then all of the target's lost Hit Points will be restored
1 then the Regen/Time parameter is the Maximum Hit Point percentage to replace each second
2 the Modifier is the amount of Hit Points to regenerate per second
3 then the Regen/Time parameter's value is the amount of seconds it takes to regenerate 1 Hit Point {% capture note %} Permanent timing modes require a duration for this effect to work. {% endcapture %} {% include important.html %}
