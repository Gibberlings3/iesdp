--- n: 98 name: "HP: Regeneration" param1: "Regeneration Amount" param2: "Type" special: "Frequency Multiplier" bg1: 0 bg2: 1 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Restores HP to the value of 'Regeneration Amount' in the style specified by the 'Type' field, to the targeted creature(s).
Known values for 'Type' are:
0 Restore 'Regeneration Amount' per second
1 Restore 'Regeneration Amount' as a percentage of total HP
2 Restore 'Regeneration Amount' HP every second
3 Restore 1HP every 'Regeneration Amount' seconds
4 Restore Param 3 HP every Param 1 seconds

This opcode uses its special field as a 'Frequency Multiplier', with a default(0) value of '1'. In particular:
  • if 'Type' is either 0 or 1, this field is unused
  • if 'Type' is 2, then Frequency = 'Frequency Multiplier'
  • if 'Type' is 3, then Frequency = 'Regeneration Amount' * 'Frequency Multiplier'
  • if 'Type' is 4, then Frequency = 'Regeneration Amount' * 'Frequency Multiplier'
{% capture note %} 'Type' values greater or equal to 5 will chrash the game! {% endcapture %} {% include warning.html %} {% capture note %} Permanent timing modes require a duration for this effect to work. {% endcapture %} {% include important.html %}
