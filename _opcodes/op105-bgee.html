Known values for 'Type' are:
0 Cumulative Modifier -> Gold = Gold - 'Statistic Modifier' value
1 Flat Value Modifier -> Gold = 'Statistic Modifier' value
2 Percentage Modifier -> Gold = (Gold * 'Statistic Modifier' value) / 100 {% capture note %} 'Type' 1 can only remove gold. Gold amount has a lower bound of 0. {% endcapture %} {% include note.html %}
