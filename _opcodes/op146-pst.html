--- n: 146 name: "Spell: Cast Spell (at Creature)" param1: "Casting Level" param2: "Type" bg1: 0 bg2: 0 bgee: 0 iwd1: 0 iwd2: 0 pst: 1 --- Casts the spell specified by the resource key at the level specified by the 'Casting Level' field, in the style specified by the 'Type' field. A 'Casting Level' of 0 casts at the level of the casting creature.
Target-type 'Global' forcs all creatures within range to cast the spell.
Known values for 'Type' are:
0 Normal Casting
1 Cast Instantly
If the target is a container, the Type parameter is ignored and treated as 0
