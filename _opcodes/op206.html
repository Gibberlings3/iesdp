--- n: 206 name: "Spell: Protection from Spell" param1: "String Reference" param2: "Irrelevant" bg1: 0 bg2: 1 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Gives the targeted creature(s) immunity from the spell specified in the resource key. The strref specified by the String Reference field is displayed when the spell is absorbed. {% capture note %} At least in EEs, this opcode can block EFF files too.
