--- n: 248 name: "Item: Set Melee Effect" param1: "Irrelevant" param2: "Fist only" bg1: 0 bg2: 1 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- {% include relurl.html %} Sets the effect specified by the resource key as a melee effect.
The specified effect will be applied to the target of each successful melee attack.
The resource key references a EFF not a SPL. {% capture note %} EFF files applied by this opcode can be blocked by opcode #101. {% endcapture %} {% include important.html %} {% capture note %} At least in EEs, setting parameter2 to 4 (i.e., BIT2) results in restricting this effect to fists only. If you leave it at 0, then all melee weapons are eligible (including fists). {% endcapture %} {% include note.html %} {% capture note %} It will always apply to both main and offhand weapon. {% endcapture %} {% include note.html %} {% capture note %} This opcode applies its effect inbetween base weapon damage and the on-hit effects of the weapon. {% endcapture %} {% include note.html %}
