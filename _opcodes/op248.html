--- n: 248 name: "Item: Set Melee Effect" param1: "Irrelevant" param2: "Fist only" bg1: 0 bg2: 1 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Sets the effect specified by the resource key as a melee effect.
The specified effect will be applied to the target of each successful melee attack.
The resource key references a .eff not a .spl.

Setting parameter2 to 4 (i.e., BIT2) results in restricting this effect to fists only. If you leave it at 0, then all melee weapons are eligible (including fists).
