The resource key references a EFF not a SPL. {% capture note %} EFF files applied by this opcode can be blocked by opcode #101. {% endcapture %} {% include important.html %} {% capture note %} At least in EEs, this opcode applies its effect after both the base weapon damage and the on-hit effects of the weapon. {% endcapture %} {% include note.html %}
