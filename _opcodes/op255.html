--- n: 255 name: "Item: Create Inventory Item (days)" param1: "Charges" param2: "Irrelevant" bg1: 0 bg2: 1 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Creates the item specified by the resource key. The number of items created is controlled by the 'Charges' field. The number of secondary charge is determined by P3. The number of tertiary charges is determined by P4 (external eff's). The item is created in the Inventory, and has the 'can not steal' bit set. {% capture note %} The difference from opcode #122 is that when used in an external .eff, the duration (from the calling file) is specified in days. {% endcapture %} {% include note.html %}
