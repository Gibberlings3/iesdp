--- n: 266 name: "Stat: Movement Modifier (forced)" param1: "Statistic Modifier" param2: "Type" bg1: 0 bg2: 0 bgee: 0 iwd1: 1 iwd2: 1 pst: 0 --- Applies the modifier value specified by the 'Statistic Modifier' field in the style specified by the 'Type' field.
Known values for 'Type' are:
0 Cumulative Modifier -> Movement = Movement + 'Statistic Modifier' value
1 Flat Value Modifier -> Movement = 'Statistic Modifier' value
2 Percentage Modifier -> Movement = (Movement * 'Statistic Modifier' value) / 100
