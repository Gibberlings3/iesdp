--- n: 282 name: "Script: Scripting State Modifier" param1: "Value" param2: "Scripting State" bg1: 0 bg2: 1 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Modifies Scripting state to the given value. Scripting State range from 0 to 35. The scripting state can be checked via scripts (see stats.ids), subtract 156 from the stat value to get the scripting state.
This effect does not have proper bounds checking, and can therefore be used in unintended ways. For example scripting state 13 equals with the NO_CIRCLE attribute (stat=169), and scripting state 1139 ties to PICKPOCKET (stat=29).
