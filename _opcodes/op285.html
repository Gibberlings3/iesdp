--- n: 285 name: "Stat: Melee Weapon Damage Modifier" param1: "Statistic Modifier" param2: "Type" bg1: 0 bg2: 1 bgee: 1 iwd1: 0 iwd2: 1 pst: 0 --- Applies the modifier value specified by the 'Statistic Modifier' field in the style specified by the 'Type' field.
Known values for 'Type' are:
0 Cumulative Modifier -> Damage = Damage + 'Statistic Modifier' value
1 Flat Value Modifier -> Damage = 'Statistic Modifier' value
2 Percentage Modifier -> Damage = (Damage * 'Statistic Modifier' value) / 100
The effect modifies stat 167.
