--- n: 321 name: "Removal: Effects specified by Resource" param1: "Irrelevant" param2: "Type" bg1: 0 bg2: 0 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Removes effects from the resource specified by the resource key, in the style specified by the Type field. Both spell and item resources are considered.
Known values for Type are:
  • 0 ⟶ Default
    • Will remove effects applied with any Timing Mode
  • 1 ⟶ Equipped effects list only
    • Will only remove effects applied with Timing Mode 2
  • 2 ⟶ Timed effects list only
    • Will remove effects applied with any Timing Mode, excluding 2 and 9
{% capture note %} It always needs to be the first effect for stacking prevention. Timing Mode should be either Instant/Limited(0) with duration=0, or Instant/Permanent(1). {% endcapture %} {% include note.html %} {% capture note %} Leaving the resource field empty will remove any effect on the creature without a parent resource. Weapon proficiencies, HLA markers, stealth, turn undead (paladin's panic only) ... Local variable effects are however safe. {% endcapture %} {% include warning.html %}
