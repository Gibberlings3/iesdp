--- n: 333 name: "Spell Effect: Static Charge" param1: "Number" param2: "Level" special: "Delay" bg1: 0 bg2: 0 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Casts a secondary spell (default: "original resource" + B) on the target, for a number of strikes specified by the 'Number' field. The spell will be cast once per 'Delay' through its projectile.
If a spell 'Level' is not specified, then the spell will be cast at level 1 and the damage opcodes will inherit the dice values of this effect, otherwise the spell will be cast at the given level.
'Delay' specifies the delay between hits, its default value is 6 (one round). {% capture note %} Its Saving Throw is checked EVERY time it triggers, terminating the op333 effect if successful, so unless that is desired (or it only triggers once), it should not have a Saving Throw. {% endcapture %} {% include note.html %}
