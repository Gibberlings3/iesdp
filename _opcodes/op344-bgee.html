--- n: 344 name: "Enchantment vs. creature type" param1: "IDS Entry" param2: "IDS File" param3: "Weapon Slot" param4: "Weapon Category" special: "Enchantment Level" bg1: 0 bg2: 0 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Alters the targeted creature(s) Enchantment against the creature type specified by the IDS Entry field in the specified IDS File.
Known values for IDS File are:
  • 2 ⟶ EA.IDS
  • 3 ⟶ GENERAL.IDS
  • 4 ⟶ RACE.IDS
  • 5 ⟶ CLASS.IDS
  • 6 ⟶ SPECIFIC.IDS
  • 7 ⟶ GENDER.IDS
  • 8 ⟶ ALIGN.IDS
  • 9 ⟶ KIT.IDS

When used in external EFFs, known values for Weapon Slot are:
  • 0 ⟶ Current weapon (the one to which this opcode is attached as an equipped effect, otherwise no restrictions)
  • 1 ⟶ Main hand weapon (includes Ammo and Fists)
  • 2 ⟶ Off-hand weapon
  • 3 ⟶ Both weapons

(External EFFs) Values for Weapon Category are taken from ITEMCAT.IDS. {% capture note %} It only sets the enchantment level, not the Magical flag. {% endcapture %} {% include note.html %}
