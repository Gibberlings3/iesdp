--- n: 345 name: "Enchantment bonus" param1: "Enchantment Level" param2: "Type" special: "Weapon Slot" bg1: 0 bg2: 0 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Known values for 'Type' are:
0 Magical weapons ('Enchantment Level' and below)
1 Magical weapons (all)
2 Non-magical weapons
3 Silver
4 Non-silver
5 Non-silver/Non-magical
6 Two-handed
7 Non two-handed
8 Cursed
9 Non-cursed
10 Cold iron
11 Non-cold iron

'Weapon Slot' can be one of the following:
0 - current weapon (the one to which this opcode is attached as an equipped effect)
1 - main hand weapon
2 - off hand weapon
3 - both weapons {% capture note %} 'Parameter4' is the item type of the affected weapon (0 matches all). This can be set to affect only fists, for example. {% endcapture %} {% include note.html %}
