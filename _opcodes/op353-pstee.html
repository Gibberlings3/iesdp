--- n: 353 opname: "Tint Playing Screen Fade" param1: "RGB Colour" param2: "Type" bg1: 0 bg2: 0 bgee: 0 iwd1: 0 iwd2: 0 pst: 0 pstee: 1 --- This effect tints the screen depending on the value specified by the 'Type' field, to a colour specified by the using 'RGB Colour' field.
The 'RGB Colour' field is handled as follows:
First byte = Red (0-255)
Second byte = Green (0-255)
Third byte = Blue (0-255)

Know values for 'Type' are:
0,1,2,3 ⟶ No effect
4,5,6,7 ⟶ Instant 'RGB Colour' for the duration of the effect
8 ⟶ No effect
9 ⟶ Instant inverted 'RGB Colour' for the duration of the effect
10 ⟶ Instant black for the duration of the effect {% capture note %} Character colour will not be affected by this effect. {% endcapture %} {% include note.html %}
