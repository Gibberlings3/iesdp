--- n: 368 opname: "Play BAM with expiration effect" param1: "Irrelevant" param2: "Flags" bg1: 0 bg2: 0 bgee: 0 iwd1: 0 iwd2: 0 pst: 0 pstee: 1 --- Applied internally by opcode 375 to control duration of the specified BAM. When the effect ends, the attached BAM is removed and a hardcoded BAM (S034DSPR.BAM) is played back once to simulate an expiration effect.
Known values for Flags are:
  • BIT6 ⟶ Blended
  • BIT19 ⟶ Grayscale
  • BIT21 ⟶ Bright
  • BIT24 ⟶ Hide background part on movement (BAM cycles used for drawing behind the target are skipped when the target is moving.)
  • BIT25 ⟶ Dream palette (sepia effect)
  • BIT27 ⟶ Garbled frames? (Garbled frames are drawn in regular intervals. Meaning of this bit is unknown.)
