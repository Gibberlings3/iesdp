--- n: 369 name: "Play BAM File Blended and Sticky" param1: "RGB" param2: "Method" bg1: 0 bg2: 0 bgee: 0 iwd1: 0 iwd2: 0 pst: 0 pstee: 1 --- Plays BAM file along with other special features. Animation is blended.
Known values for Method are:
  • BIT0 ⟶ Repeat animation, otherwise duration counts, except for permanent which plays once.
  • BIT1 ⟶ Removes stickiness (background animation)
