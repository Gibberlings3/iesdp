STO file format

Applies to:
BG1, BG1: TotS, BG2, BG2: ToB, IWD, BGEE

General Description
These files contain a description of the types of items and services available for sale in a given store, inn, tavern, or temple.

    STOR V1.0 Items Purchased Here

    This is an array of dwords. These dwords indicate categories of items from the Item Types table.

    Item Types

    These codes are used in store files to indicate which types of items they will buy. They are also used to determine which inventory slots the item is allowed to go into on the inventory screen and the sound the item makes when moved in the inventory.
