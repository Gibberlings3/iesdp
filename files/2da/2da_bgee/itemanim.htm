itemdial.2da

This file lists the character animation sequence to be played when an item ability is used.
In BG2EE we have:
2DA V1.0
-1
           SEQUENCE   
POTN00     -1         
FIRESEED   8
The first column is the item reference.
The second column is the animation sequence to play (from seq.ids).
