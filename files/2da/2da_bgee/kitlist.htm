kitlist.2da

This file defines kit-related information.
The first column contains numeric identifiers for each kit.
The second column contains character identifiers for each kit.
The third column lists a value of a strref, which contains the kit name in lowercase (eg. berserker).
The fourth column lists a value of a strref, which contains the kit name in mixed case (eg. Berserker).
The fifth column lists a value of a strref, which contains a summary description of the kit.
The sixth column contains the filename of the clab to be used when levelling up the kit.
The seventh column is a column index into weapprof.2da.
The eighth column contains a valae relating to the items the kit is allowed to use. These values correspond to the kit usability flags in itm files.
The ninth column lists identifiers, controlling which class selection screen the kit appears under (and also some main kit restrictions, such as the buttons in the button bar).
The tenth column defines a value that is referenced in KIT.IDS which can be referred to in script actions or triggers.
