savecnxx.2da

This file defines a constitution bonus to saving throws for
- Dwarves and Halflings: savecndh.2da
- Gnomes: savecng.2da

The first row (titles of columns) is filled with the individual constitution scores.
The first column defines the saving throw type.
The other columns define the saving throw bonuses for the given constitution score.
