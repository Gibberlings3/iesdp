stweapon.2da

This file defines the starting weapon of new characters in BG1.
The first column defines the name of the class/multiclass.
The second column defines the weapon that should be created in the first weapon slot. Leave empty to create no weapon at all.
In BGEE we have:
2DA V1.0
STAF01
           WEAPON
MONK
