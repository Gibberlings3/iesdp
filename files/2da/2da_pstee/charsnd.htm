charsnd.2da

This file ties sound files to their string references, to be displayed as subtitles.
The first column shows the sound slot each row is associated with. Values are from SNDSLOT.IDS.
The second column shows the string reference for a male character.
The third column shows the string reference for a female character.
Subsequent columns show the string references for custom soundsets.
