This file defines the starting equipment of new characters in the ToB expansion.
The first column defines the slot where given item is placed - its name
Second , labeled by ID is UNKNOWN
The next two columns reference to the specified strrefs from Dialog.tlk to describe the slot .
NAME_REF
DESC_REF
The next columns are labeled by names of the kits and classes/multiclasses.
The crosses of these names and names of slots in first column are filled by the filenames of ITM files . If the item is countable then after the comma there is number indicating the amount of the given ITM in the slot (for potions, arrows for example)
The asterisk in place of crossing indicates that the slot is not filled by anything in this case.
