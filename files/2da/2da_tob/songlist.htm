songlist.2da

This file assigns the MUS file to the certain values. and is used by scripting and ARE files to determine what music should be played. Only available in BG2 SoA / ToB.
The first column is filled by numbers used by ARE files song sections and scripting commands like PlaySong(number).
The second one contains of descriptions /identifiers (I have no idea if they can be used instead of values in scripts).
And third one contains of MUS files references that describes what ACM sound files are used and how to play song.
The maximum number of entries is 100 (ie. 0 to 99).
