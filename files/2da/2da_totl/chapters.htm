chapters.2da

This file determines the text displayed on the chapter intermission screens when called by the action IncrementChapter.
The first column is matched to the current value of the global variable "Chapter". The second column is the strref used for the title.
The third column is the strref used for the scrolling text.
The fourth column is unused.
