efftext.2da

This file defines names of states (like death, poisoned etc....) that are shown in the game message window.
The first column consists of number of the effect, tested it with some effects like slow (40) and Charm Specific Creature (5) and others.
Second column is as it is said - the effect description without effect on game - just for information.
Third column consists of the strref with the name of this effect that is shown in game.
